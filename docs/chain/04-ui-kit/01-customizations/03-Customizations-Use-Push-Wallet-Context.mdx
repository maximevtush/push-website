The `usePushWalletContext` hook provides access to the current wallet state and methods to interact with the Push Wallet. This hook must be used within a component that's wrapped by a `PushUniversalWalletProvider`.


## Usage

```jsx live
// customPropHighlightRegexStart=const WalletContextComponent
// customPropHighlightRegexEnd=</PushUniversalWalletProvider>
import { 
  PushUniversalWalletProvider,
  PushUniversalAccountButton,
  usePushWalletContext,
  PushUI 
} from '@pushchain/ui-kit';
import { useState, useEffect } from 'react';

// Create a component that uses the hook inside the provider context
const WalletContextComponent = () => {
  const { 
    account, 
    isConnected, 
    isConnecting, 
    connect, 
    disconnect 
  } = usePushWalletContext(); // optional: uid parameter for targeting a specific wallet instance

  return (
    <div>
      <h3>Wallet Status</h3>
      <p>Connected: {isConnected ? 'Yes' : 'No'}</p>
      <p>Connecting: {isConnecting ? 'Yes' : 'No'}</p>
      {isConnected && account && (
        <p>Account: {account.address}</p>
      )}
      {!isConnected && (
        <button onClick={() => connect()}>Connect Manually</button>
      )}
      {isConnected && (
        <button onClick={() => disconnect()}>Disconnect</button>
      )}
    </div>
  );
};

function App() {
  return (
    <PushUniversalWalletProvider config={{ network: PushUI.CONSTANTS.PUSH_NETWORK.TESTNET }}>
      <PushUniversalAccountButton />
      <WalletContextComponent />
    </PushUniversalWalletProvider>
  );
}
```

### Parameters

| Arguments | Type     | Required | Description                                 |
| --------- | -------- | -------- | ------------------------------------------- |
| `uid`     | `string` | No       | Optional user identifier for wallet context |

#### Return Values

| Property                    | Type     | Description                                                                                                                             |
| --------------------------- | -------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| `connectionStatus`          | string   | Current connection state of the wallet. Possible values: `"notConnected"`, `"connecting"`, `"authenticating"`, `"connected"`, `"retry"` |
| `handleConnectToPushWallet` | function | Call this function to initiate the wallet connection process                                                                            |
| `handleUserLogOutEvent`     | function | Call this function to disconnect the current wallet and log out the user                                                                |
