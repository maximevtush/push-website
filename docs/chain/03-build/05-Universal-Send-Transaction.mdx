## Overview

Universal transactions let you send **native** transactions from any Layer 1 chain‚ÄîEVM or non-EVM, even Push Chain itself‚Äîwithout wrapping, bridging or extra tooling required.

Behind the scenes, the SDK automatically estimates gas and assembles the necessary cross-chain signature/verification, so you can focus on your app instead of network details.

## Sending Universal Transaction
**_`pushChainClient.universal.sendTransaction({options}): Promise<TransactionReceipt>`_**

```typescript
const txHash = await pushChainClient.universal.sendTransaction({
  to: '0xa54E96d3fB93BD9f6cCEf87c2170aEdB1D47E1cF',
  value: BigInt('100000000000000000'), // 0.1 PC in wei
});
```

| **Arguments**          | **Type** | **Description**                                       |
| ---------------------- | -------- | ----------------------------------------------------- |
| _`options.to`_                 | `string` | The address of the recipient.                         |
| _`options.value`_              | `BigInt` | The value to send in wei.                             |
| `options.data`                 | `string` | The data to send.                                     |
| `options.gasLimit`             | `BigInt` | The gas limit for the transaction.                    |
| `options.maxFeePerGas`         | `BigInt` | The maximum fee per gas for the transaction.          |
| `options.maxPriorityFeePerGas` | `BigInt` | The maximum priority fee per gas for the transaction. |
| `options.deadline`             | `BigInt` | The deadline for the transaction.                     |

> Note: These _`Arguments`_ are mandatory.

## Send Transaction with Contract Interaction

When you need to interact with smart contracts, you'll need to pass encoded function data. Here's how to generate the `data` parameter and use it with the Push Chain SDK:

<Tabs groupId="send-transaction">
  <TabItem value='ethers_v6' label='Ethers (v6)' default>

```typescript
import { ethers } from 'ethers';

async function main() {
  // Define the ABI for the ERC20 transfer function
  const erc20Abi = [
    'function transfer(address to, uint256 amount) returns (bool)',
  ];

  // Create an Interface instance
  const interface = new ethers.Interface(erc20Abi);

  // Encode the function data
  const data = interface.encodeFunctionData('transfer', [
    '0xRecipientAddress',
    ethers.parseEther('10'),
  ]);

  // Send the transaction using Push Chain SDK
  const txHash = await pushChainClient.universal.sendTransaction({
    to: '0xTokenContractAddress', // The smart contract address on Push Chain
    value: 0n, // No ETH being sent
    data: data, // Encoded function call
  });
}

main().catch(console.error);
```

  </TabItem>

  <TabItem value='viem' label='Viem'>

```typescript
import { encodeFunctionData, parseEther } from 'viem';

async function main() {
  // Define the ABI for the ERC20 transfer function
  const erc20Abi = [
    'function transfer(address to, uint256 amount) returns (bool)',
  ];

  // Generate the encoded function data using viem
  const data = encodeFunctionData({
    abi: erc20Abi,
    functionName: 'transfer',
    args: ['0xRecipientAddress', parseEther('10')] // Transfer 10 tokens
  });

  // Send the transaction using Push Chain SDK
  const txHash = await pushChainClient.universal.sendTransaction({
    to: '0xTokenContractAddress', // The smart contract address on Push Chain
    value: BigInt('0'), // No ETH being sent, just contract interaction
    data: data // The encoded function call
  });
}

main().catch(console.error);
```
